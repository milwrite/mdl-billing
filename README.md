# Muhlbauer Dermatopathology Billing POC

Interactive flow-chart proof of concept showing synthetic insurance claim routing pathways for dermatopathology billing.

## What it is

A static website that demonstrates how claims may move through:
- insurer rebilling
- patient ledger
- collections
- manager review / negotiated reduction
- low-yield payer awareness

## What it is not

- not production billing logic
- not real patient data
- not connected to a live PM/LIS/clearinghouse

## Sources

Built from:
- internal transcript summary and memo
- research on pathology / laboratory denial handling, claim scrubbing, and appeals workflows
